# A股龙虎榜数据

## 数据说明

本目录包含A股市场龙虎榜历史数据，数据来源于东方财富网数据中心。

## 数据文件

### 龙虎榜详情_2020至今.csv

包含2020年1月1日至今的所有龙虎榜详情数据，共计约11万条记录。

| 字段名称 | 数据类型 | 说明 |
|---------|---------|------|
| 交易日期 | datetime | 上榜交易日期 |
| 代码 | string | 股票代码 |
| 名称 | string | 股票简称 |
| 收盘价 | float | 当日收盘价 |
| 涨跌幅 | float | 当日涨跌幅(%) |
| 龙虎榜净买额 | float | 龙虎榜净买入金额(元) |
| 龙虎榜买入额 | float | 龙虎榜买入金额(元) |
| 龙虎榜卖出额 | float | 龙虎榜卖出金额(元) |
| 龙虎榜成交额 | float | 龙虎榜成交金额(元) |
| 市场总成交额 | float | 当日市场总成交额(元) |
| 净买额占总成交比 | float | 净买额占总成交比例(%) |
| 成交额占总成交比 | float | 成交额占总成交比例(%) |
| 换手率 | float | 当日换手率(%) |
| 流通市值 | float | 流通市值(元) |
| 上榜原因 | string | 上榜原因说明 |
| 上榜后1日涨跌幅 | float | 上榜后第1个交易日涨跌幅(%) |
| 上榜后2日涨跌幅 | float | 上榜后第2个交易日涨跌幅(%) |
| 上榜后5日涨跌幅 | float | 上榜后第5个交易日涨跌幅(%) |
| 上榜后10日涨跌幅 | float | 上榜后第10个交易日涨跌幅(%) |

## 数据来源

- 数据来源：东方财富网数据中心
- 原始数据地址：https://data.eastmoney.com/stock/tradedetail.html
- 数据更新时间：2026年2月2日

## 使用说明

```python
import pandas as pd

# 读取数据
df = pd.read_csv('龙虎榜详情_2020至今.csv')

# 查看数据概览
print(df.info())
print(df.head())

# 按日期筛选
df['交易日期'] = pd.to_datetime(df['交易日期'])
df_2024 = df[df['交易日期'].dt.year == 2024]
```
