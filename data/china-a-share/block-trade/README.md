# A股大宗交易数据

## 数据说明

本目录包含A股市场大宗交易历史数据，数据来源于东方财富网数据中心。

## 数据文件

### 1. 大宗交易市场统计_2020至今.csv

包含2020年至今的大宗交易市场统计数据，记录每日大宗交易市场整体情况。

| 字段名称 | 数据类型 | 说明 |
|---------|---------|------|
| 交易日期 | datetime | 交易日期 |
| 大宗交易成交总额 | float | 当日大宗交易成交总额(元) |
| 溢价成交总额 | float | 溢价成交总额(元) |
| 折价成交总额 | float | 折价成交总额(元) |
| 成交笔数 | int | 当日成交笔数 |
| 溢价成交笔数 | int | 溢价成交笔数 |
| 折价成交笔数 | int | 折价成交笔数 |
| 上证指数 | float | 当日上证指数收盘价 |
| 上证指数涨跌幅 | float | 当日上证指数涨跌幅(%) |

### 2. 大宗交易每日统计_2020至今.csv

包含个股大宗交易每日统计数据。

| 字段名称 | 数据类型 | 说明 |
|---------|---------|------|
| 序号 | int | 序号 |
| 交易日期 | datetime | 交易日期 |
| 证券代码 | string | 股票代码 |
| 证券简称 | string | 股票简称 |
| 涨跌幅 | float | 当日涨跌幅(%) |
| 收盘价 | float | 当日收盘价 |
| 成交价 | float | 大宗交易成交价 |
| 折溢率 | float | 折溢率(%) |
| 成交笔数 | int | 成交笔数 |
| 成交总量 | float | 成交总量(股) |
| 成交总额 | float | 成交总额(元) |
| 成交总额/流通市值 | float | 成交总额占流通市值比例(%) |

## 数据来源

- 数据来源：东方财富网数据中心
- 原始数据地址：https://data.eastmoney.com/dzjy/
- 数据更新时间：2026年2月2日

## 使用说明

```python
import pandas as pd

# 读取市场统计数据
df_market = pd.read_csv('大宗交易市场统计_2020至今.csv')

# 读取每日统计数据
df_daily = pd.read_csv('大宗交易每日统计_2020至今.csv')

# 查看数据概览
print(df_market.info())
print(df_daily.head())

# 分析大宗交易与指数关系
df_market['交易日期'] = pd.to_datetime(df_market['交易日期'])
```
