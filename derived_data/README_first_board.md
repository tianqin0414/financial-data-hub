# 首板打板策略回测数据说明

## 数据文件

### 1. tradingview_a_stock_daily_20241101_20250514.csv

**来源**: TradingView  
**时间范围**: 2024-11-01 至 2025-05-14  
**数据量**: 804,143 行 × 138 列  
**文件大小**: 1.3 GB（未上传到仓库，因超过 GitHub 限制）  
**说明**: A股日线数据，包含价格、技术指标、企业性质等信息

> ⚠️ **注意**: 原始数据文件过大（1.3GB），未上传到 GitHub。如需原始数据，请联系数据提供者或使用 TradingView 自行导出。

**关键字段**:
- `数据日期`: 交易日期
- `商品代码`: 股票代码
- `描述`: 股票名称
- `涨跌幅%`: 当日涨跌幅
- `第2日盘前涨跌幅%`: 次日高开幅度
- `企业性质`: 企业类型（地方国有企业、中央国有企业、大型民企等）
- `近20日涨幅`、`近30日涨幅`、`近40日涨幅`: 计算得出的近期涨幅

### 2. first_board_backtest_results.csv

**说明**: 首板打板策略参数网格搜索结果

**字段**:
- `近期天数`: 近期涨幅计算天数（20/30/40）
- `最大涨幅`: 近期最大涨幅阈值（%）
- `最小高开`: 最小高开幅度（%）
- `企业类型`: 企业类型过滤条件
- `交易次数`: 触发信号次数
- `胜率`: 次日盈利的概率（%）
- `平均收益`: 次日平均收益率（%）
- `总收益`: 累计收益率（%）
- `最大回撤`: 最大回撤（%）

### 3. first_board_best_signals.csv

**说明**: 最优策略（近30日涨幅≤15%、高开≥4%、地方国企）的所有交易信号

**字段**: 包含原始数据的所有字段，以及：
- `return`: 次日收益率（%）

### 策略实现

```python
def is_first_board_signal(row):
    """
    判断是否触发首板打板信号
    """
    return (
        row['近30日涨幅'] <= 15 and  # 近30日涨幅不超过15%
        row['第2日盘前涨跌幅%'] >= 4.0 and  # 高开幅度≥4%
        row['涨跌幅%'] >= 9.5 and  # 当日涨停
        row['企业性质'] == '地方国有企业'  # 地方国企
    )
```
